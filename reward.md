# JToken
- **[JToken](https://jtoken.onrender.com/)** is a token-based reward system that can be redeemed for extra credits.

## How to earn JTokens?
- Attendance = 1 JToken
- Recitation = 1 JToken
- Lab Activity = 1 JToken
- Programming Competition = 5 JTokens
  
## JToken Value
- Prelim = 5 points
- Midterm = 4 points
- Pre-finals = 3 points
- Finals = 2 points
